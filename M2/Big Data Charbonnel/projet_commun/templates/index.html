Comment interroger l’encyclopédie ?

Pour récupérer tous les données disponible dans la base de donnée locale :

L’URL /ws renvoie toutes les données sous forme de dictionnaire

Pour récupérer les topics de l’encyclopédie :

L’URL /ws/topics renvoie la liste des topics disponibles localement. Par exemple http://12I008.stud.ua:8888/ws/topics pourrait renvoyer : 

[ "types", "fonctions", "multithreading", "POO" ]

Pour récupérer les items d’un topic donné :

L’URL /ws/topic/le_topic (où le_topic est un argument à fournir) renvoie un dictionnaire dont les clés sont les items du topic, et la valeur une liste d’URL. Par exemple http://12I008.stud.ua:8888/ws/topic/multithreading renverrait : 

{"exemples": [url, url,...], "piège": [url, url,...], "FAQ": [url, url,...], "définition": [url, url,...]}

Pour récupérer les adresse des serveurs qui participe au projet :

Chaque service web fournit, en réponse à la requête /ws/annuaire, la liste des serveurs (qu’il connait) participant à l’encyclopédie, sous la forme d’une liste d’URL. 

[ "http://01i008:8888", "http://03i008:1245", "http://192.168.123.10:2000"]
